# 重构的刻意练习 [![][Badges: Travis CI]][Links: Travis CI]

再版序中，大🐻已经表达得很清晰，学习重构，抓 **坏味道**、**对应的重构手法**、**遵循特定步骤**。是学习线索。当熟读此书。
